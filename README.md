# Maqueta Habit Tracker con Análisis Diario, Semanal y Mensual

## Descripción
Este proyecto permite registrar hábitos diarios y generar análisis de progreso en tres niveles:
- **Diario**: seguimiento puntual de cada hábito.
- **Semanal**: resumen de tendencias y consistencia.
- **Mensual**: estadísticas globales y comparativas.

## Objetivo
Ayudar al usuario a mantener hábitos saludables y productivos mediante visualizaciones claras y reportes automáticos.
